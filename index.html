What is CatKernel?

CatKernel is a Unix-like kernel; made to be fast, versatile, and useful. It shares some similarity in layout & structure to BSD Unix.

The 'Bleeding-Edge' Disk Image has the most recent, testing builds of CatK and other binaries, all on the Disk. It is what we use when testing, or porting over new applications. The CatK 'Stable' Disk Image has known stable, working Binaries, and Kernel. Both Disk Images are in i386 compatible binary format.

Who is maintaining CatKernel?

CatKernel is being actively worked on by the CatKernel team at the Github repository. If you have any issues, please report it to the Issues section of the repository.


How can I contribute?

You can help via developing kernel drivers, programs, or the main kernel itself. Please contact us at the Discord server, It's quite nice there. Or directly request changes to merge at Github repository.
